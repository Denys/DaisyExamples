═══════════════════════════════════════════════════════════════════════════════════════════
SIGNAL PATH SUMMARY
═══════════════════════════════════════════════════════════════════════════════════════════

MIDI/Keyboard → Note Processor → Voice Manager (8 voices)
                                       ↓
                                  Each Voice:
                                  OP1-4 (Fm2) → ADSR Env → Algorithm → SVF Filter
                                       ↓
                                  Voice Mixer (Sum ÷ 8)
                                       ↓
                                  CHORUS EFFECT
                                  - Delay Line 1 (7ms + LFO)
                                  - Delay Line 2 (13ms + LFO phase shifted)
                                  - Dry/Wet Mix
                                       ↓
                                  Limiter (Safety)
                                       ↓
                                  Stereo Spread (L/R)
                                       ↓
                                  Audio Output


═══════════════════════════════════════════════════════════════════════════════════════════
CHORUS PARAMETERS
═══════════════════════════════════════════════════════════════════════════════════════════

Delay Line 1:        7ms base delay
Delay Line 2:        13ms base delay
LFO Rate:            0.5 - 2.0 Hz (CV2 controlled)
LFO Depth:           0 - 3ms (CV1 controlled)
LFO Phase Shift:     90° between lines (for stereo width)
Dry/Wet Mix:         0 - 100% (CV3 controlled)
Stereo Spread:       Delay 1 → Left, Delay 2 → Right
