# Pod_DrumMachinePro

6-voice, 16-step drum machine for Daisy Pod.

## Controls

- `Encoder turn`: Select edit step (1-16)
- `Encoder press`: Cycle parameter page (1-4)
- `Button 1`: Toggle selected step for current drum
- `Button 2`: Cycle drum voice
- `Knob 1/2`: Edit active page parameters

## Parameter Pages

- `Page 1`: Kick Decay / Snare Decay
- `Page 2`: CHat Decay / OHat Decay
- `Page 3`: Tom Tune / Clap Decay
- `Page 4`: Tempo / Swing

## Build

```bash
cd DaisyExamples/MyProjects/_projects/Pod_DrumMachinePro
make clean
make
```
