Load and run the preset loader first (pollen8_VA_FM_field_presetLoader1.bin), 
as it will erase any you create before loading them. You will be prompted to hold
SW1 and SW2 to begin the loading process after running. 

pollen8_VA_FM_field_v2_0.bin contains the binary for an electro-smith Daisy Field
synthesizer, you can download this using their web programmer at 
https://electro-smith.github.io/Programmer/
- note this currently requires google chrome version 61 or above.
