Daisy Field drum-machine UI replica

Replica of the Daisy Field front panel with the suggested OXI-assisted control split. Base mode keeps the Field focused on voice selection, status, and sound shaping. SW1 and SW2 expose step and performance layers without permanently consuming the 16 key LEDs.

LED off = absent / idle / not armed
LED on = selected / latched / stored
LED blink = temporal event / playhead / active fill

Current layer

8 encoders

Recommended split with OXI One

OXI One: clock, start/stop, long pattern editing, probability, ratchets, microtiming, song/pattern structure.
Daisy Field: track focus, macro performance, sound pages, quick step overlay for current track, kit navigation, local status.
SW1 = Shift / step overlay.
SW2 = Alt / performance layer.
Tip: click the mode buttons or use the on-panel SW1/SW2 buttons. Hover over a pad for the assigned function. This replica is intentionally not a pixel-perfect CAD drawing; it is a UI architecture mockup anchored to the Daisy Field front layout.
