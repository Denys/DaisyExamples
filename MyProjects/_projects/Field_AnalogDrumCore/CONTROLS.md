# Controls - Field_AnalogDrumCore

## Knobs

| Knob | Parameter | Range | Default | Notes |
|------|-----------|-------|---------|-------|
| K1 | Kick Tone | 0-1 | 0.65 | Brightness/click of kick |
| K2 | Kick Decay | 0.05-0.95 | 0.55 | Kick tail length |
| K3 | Kick Attack FM | 0-1 | 0.35 | Pitch attack punch |
| K4 | Snare Tone | 0-1 | 0.55 | Snare body brightness |
| K5 | Snare Decay | 0.05-1.0 | 0.45 | Snare ring length |
| K6 | Snare Snappy | 0-1 | 0.60 | Snare wire/noise amount |
| K7 | Tempo | 60-240 BPM | 122 BPM | 16th-note sequencer clock |
| K8 | Drive | 0-1 | 0.35 | Master overdrive amount |

## Keys - Row A (Top)

| Key | Function |
|-----|----------|
| A1-A8 | Toggle steps 1-8 for selected lane |

## Keys - Row B (Bottom)

| Key | Function |
|-----|----------|
| B1-B8 | Toggle steps 9-16 for selected lane |

## Switches

| Switch | Function |
|--------|----------|
| SW1 | Toggle edit lane: Kick/Snare |
| SW2 | Start/Stop sequencer |

## OLED Display

| Screen | Content |
|--------|---------|
| Compact | 8 macro parameters with active highlighting |

## Presets

| Preset | Description | Key Settings |
|--------|-------------|--------------|
| Init | 4-on-floor kick + backbeat snare | Default startup pattern |
