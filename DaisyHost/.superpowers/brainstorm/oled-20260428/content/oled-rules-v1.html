Recommended baseline: B plus selected C details

  • Meaning: Q 12-JI means quantize mode is 12-note Just Intonation. In status, use Quant 12-JI; in zoom, spell it as 12-note Just.
  • Zoom rule: any K/CV/menu/key interaction owns the OLED for 2.0 seconds after the last change.
  • Knob example: K8 Cutoff, large current frequency, normalized bar, then return to status.
  • Button example: B7 Play, large Running or Stopped, with current step and quantize context.
  • Audio bug rule: if output energy collapses while playing and not intentionally muted, show Audio: silent and add a regression test for the exact trigger path.
