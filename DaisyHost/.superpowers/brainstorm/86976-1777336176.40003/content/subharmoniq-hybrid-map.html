Sound Pages: Hardware-Faithful Knob Layers

VCO: VCO1, VCO2, V1 Sub1, V1 Sub2, V2 Sub1, V2 Sub2, Quantize, Seq Oct.

VCF: Cutoff, Resonance, VCF EG Amt, VCF Attack, VCF Decay, Drive, Output, Tempo.

VCA/Mix: six source levels plus VCA Attack/Decay or Output/Drive depending available core controls.

Rule: K1-K8 changes per page; A/B keys stay performance-stable.

A

Stable A/B Keys Across Pages

Recommended. The button matrix is always sequencer/rhythm/transport, while knobs follow VCO, VCF, VCA/Mix, or Seq pages.

B

Page-Sensitive A/B Keys

More direct functions per page, but the same physical key changes meaning on different pages.
