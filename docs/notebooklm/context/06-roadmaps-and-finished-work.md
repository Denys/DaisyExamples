## How To Read Plans In This Repo

- plans capture intent and architecture
- plans do **not** prove implementation is current
- always confirm against:
  - local source
  - local docs
  - recent verification notes

## Strategic Rule

Finished work should be remembered as reusable precedent.

Planned work should be remembered as direction, not truth.
